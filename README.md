<h1 align="center">Cook It</h1>

## Project Description  

**Cook It** is a mobile/web application that helps you find recipes using the ingredients you already have at home.  

You simply enter the ingredients you have available, and the system displays recipes submitted by other users that use those ingredients.  

The application also allows anyone to add new recipes and share their kitchen ideas.  

---

## Technologies Used  

### Front-end
- HTML5  
- CSS3  
- JavaScript  
- React  
- Axios  
- React Router  

### Back-end
- Java  
- Spring Boot  
- JPA / Hibernate  
- Flyway (database versioning)  
- JUnit  
- Spring Security  
- JWT (JSON Web Token)  

### Database
- PostgreSQL  

---

## API Documentation
The API documentation is available through **Swagger UI**, which allows you to visualize and test all available endpoints in an interactive interface.  

After running the application, access:
    ```http://localhost:8080/swagger-ui.html```
